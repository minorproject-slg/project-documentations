Phase 1 MVP · Microservices

College
Management
System

A modular, scalable web-based platform to digitize academic and administrative operations. Built with Spring Boot microservices, secured by JWT, and persisted via MySQL.

Spring Boot MySQL Spring Data JPA Hibernate ORM JWT · RBAC REST APIs
System design
Backend Architecture

Five independently deployable microservices, each with an isolated MySQL database, bridged through Spring Data JPA and Hibernate ORM.

Client Layer
Admin Portal
College staff · Full access
Faculty Portal
Teaching staff · Subject access
Student Portal
Enrolled students · View access
Gateway / Security Layer
REST API
JWT Authentication
RBAC Authorization
Microservices Layer
Auth Service
Login · JWT · BCrypt
User Service
Profiles · Roles
Academic Service
Courses · Marks · GPA
Finance Service
Fees · Transactions
Notification
Alerts · Emails
Persistence Layer
Spring Data JPA · Hibernate ORM · Entity mapping · JPQL · Transactions
Data Layer — MySQL (isolated per service)

MySQL
users · roles · tokens
